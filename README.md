# 🚀 DISNOU PDV - Frente de Caixa & Estoque

O **DISNOU PDV** é um sistema completo de frente de caixa (PDV) e gerenciamento de estoque, desenvolvido para oferecer agilidade, precisão e uma experiência de usuário moderna em ambientes de varejo.

Este projeto foi construído utilizando **JavaScript Vanilla (Puro)**, focando em performance, lógica de programação e uma arquitetura limpa, sem a necessidade de bibliotecas externas pesadas.

## 💡 O Problema que este projeto resolve
Sistemas de caixa convencionais muitas vezes são lentos, possuem interfaces obsoletas e dificultam a integração real com o estoque. O DISNOU PDV foi desenhado para:
*   **Agilidade Operacional:** Operação focada no teclado (atalhos), reduzindo o tempo de atendimento.
*   **Controle em Tempo Real:** Integração imediata entre a venda finalizada e a baixa automática no inventário.
*   **Gestão Inteligente:** Painel administrativo para auditoria financeira e controle de promoções dinâmicas.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica e acessível.
- **CSS3:** Estilização responsiva com design *Dark Mode* moderno (Cyberpunk Corporativo).
- **JavaScript (ES6+):** Lógica de negócios, manipulação do DOM e persistência de dados.
- **LocalStorage:** Persistência de dados local, garantindo um sistema funcional de alta performance.

## 🚀 Funcionalidades Principais
- **Checkout Inteligente:** Adição de itens via código de barras/teclado com suporte a busca de preços.
- **Gestão de Estoque:** Painel administrativo com status de níveis de produtos (Ok, Baixo, Zerado).
- **Promoções Diárias:** Motor de regras de negócio para descontos automatizados por dia da semana.
- **Clube Fidelidade:** Sistema de desconto progressivo para clientes cadastrados.
- **Auditoria Financeira:** Histórico detalhado de vendas com segregação por formas de pagamento (Dinheiro, Cartão, PIX).
